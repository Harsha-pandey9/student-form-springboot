üîê Authentication API - /api/auth

POST /api/auth/register
Register a new user with JSON payload
POST /api/auth/login
Login and receive JWT access token
GET /api/auth/profile
Get current user profile (requires auth)
POST /api/auth/refresh
Refresh access token using refresh token
POST /api/auth/logout
Logout and invalidate tokens
GET /api/auth/validate
Validate current token
GET /api/auth/health
Health check for auth service

üìö Student API - /api/students

GET /api/students
List all students (role-based access)
GET /api/students/{id}
Get student by ID
GET /api/students/rollno/{rollNo}
Get student by roll number
GET /api/students/search?name={name}
Search students by name (Admin/Teacher only)
GET /api/students/branch/{branch}
Get students by branch (Admin/Teacher only)
GET /api/students/course/{course}
Get students by course (Admin/Teacher only)
POST /api/students
Create new student (Admin only)
PUT /api/students/{id}
Update student (Admin only)
PATCH /api/students/{id}
Partial update student (Admin only)
DELETE /api/students/{id}
Delete student (Admin only)
